html?

HTML, otherwise known as HyperText Markup Language, is the language used to create Web pages

Using HTML, you can create a Web page with text, graphics, sound, and video

html tag

The essence of HTML programming is tags.
A tag is a keyword enclosed by angle brackets.

There are opening and closing tags for many but not all tags; The affected text is between the two tags.
The opening and closing tags use the same command except the closing tag contains and additional forward slash / 


    <h1> Ini Contoh ya! </h1>
    
    <p> Halo! nama saya kevin. </p>
    <br>
    <p> Siapa nama kamu? </p>

Nested Tags

Whenever you have HTML tags within other HTML tags, you must close the nearest tag first 

    <a> <h1> ini di dalem </h1> </a>

Structure of a Web Page

Whenever you have HTML tags within other HTML tags, you must close the nearest tag first

The Title Tag

Choose the title of your Web page carefully; The title of a Web page determines its ranking in certain search engines 

    <title> Neodroid Basic Html </title>

Text Formatting

Manipulating text in HTML can be tricky; Oftentimes, what you see is NOT what you get

For instance, special HTML tags are needed to create paragraphs, move to the next line, and create headings 

    <h1> Lorem, ipsum. </h1>
    <br>
    <p> Lorem ipsum dolor sit amet. </p>

Text Formatting Tags

    <B> Lorem, ipsum. </B>
    <I> Lorem, ipsum. </I>
    <U> Lorem, ipsum. </U>
    <P> Lorem, ipsum. </P>
    <BR>

Changing the Font

The expression can be used to change the font of the enclosed text 

    <font face="fontname"> Lorem, ipsum. </font>

    <font size=n> Lorem, ipsum. </font>

    <font color="red"> Lorem, ipsum. </font>

    <font face="Courier" size=4 color="red"> 
    Lorem, ipsum. </font>

Headings

Web pages are typically organized into sections with headings; To create a heading use the expression …. where n is a number between 1 and 7

In this case, the 1 corresponds to the largest size heading while the 7 corresponds to the smallest size 

    <Hn> Lorem, ipsum. </Hn>

    <h1> Lorem, ipsum. </h1>
    <h2> Lorem, ipsum. </h2>

Aligning Text

The ALIGN attribute can be inserted in the P and Hn tags to right justify, center, or left justify the text 

    <h1 align=center> Lorem, ipsum. </title>

Comment Statements

Comment statements are notes in the HTML code that explain the important features of the code; The comments do not appear on the Web page itself but are a useful reference to the author of the page and other programmers 

    <!--> ini ga akan kebaca <-->

Inserting Images

Type this, where image.ext indicates the location of the image file

The WIDTH=n and HEIGHT=n attributes can be used to adjust the size of an image

The attribute BORDER=n can be used to add a border n pixels thick around the image 

    <img src="image.ext"> 

    <img src="image.ext" width=10 height=10> 

    <img src="image.ext" border=1>

Alternate Text

Some browsers don’t support images. In this case, the ALT attribute can be used to create text that appears instead of the image. 

<img src="image.ext" alt="picture of satellite">
